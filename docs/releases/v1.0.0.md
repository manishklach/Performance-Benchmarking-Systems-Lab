# AMD APU Toolkit v1.0.0

First public release.

## Highlights

- Browser dashboard with live GPU Focus, CPU Focus, RAM, GPU/CPU, and Overview views
- Desktop GUI with compact layouts, chart exports, and persisted thresholds
- Terminal dashboard with PNG snapshot export
- GPU process tracer sorted by descending utilization
- UMA pressure inspection for AMD APU shared-memory behavior
- OpenCL runtime probe and power correlation CSV export

## Notes

- Built for Windows-first AMD APU systems with integrated Radeon graphics
- Uses local Windows counters and local services only
- Does not require ROCm

## Known limitations

- GPU attribution is process-level rather than true thread-level GPU time
- Vendor-specific telemetry such as clocks, temperature, and power draw is not included yet
- Counter availability varies by Windows build, driver version, and hardware generation
